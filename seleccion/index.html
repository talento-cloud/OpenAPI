Talento Cloud - API de Selección (1.3.0)

Download OpenAPI specification:

Documentación de los endpoints del módulo de Selección (Requisiciones y Procesos).

Flujo de uso del panel de filtros

Antes de llamar al endpoint principal, la UI ejecuta en paralelo:

  1. GET /seleccion/propiedades/estadoCamposOcultablesRequisicion para determinar qué campos y filtros deben mostrarse u ocultarse.

  2. Los endpoints de catálogo: /sedesPaginado, /ConsultaEmpresasSpring/{negocio}, /cargosold/getPagin, /ConsultaUsuariosSpring/{negocio}/{incluirCorporativo}, /seleccion/plantillaconfiguracion/etapas-proceso/listar/etapas/{codigoPlantilla}/{listarEtapasSeleccionadas}, y /seleccion/aprobacion/flujosAprobacionAll.

  3. Con las selecciones del usuario se construye el parámetro filtrosAdicionales y se invoca GET /requisicion/listarRequisicionesEnCursoPorResposable.

Manejo de errores

Código Descripción
400 Parámetros faltantes o con formato inválido.
401 Token JWT inválido, expirado o ausente.
403 El usuario autenticado no tiene permisos suficientes.
404 El recurso solicitado no existe.
500 Error inesperado en el servidor.

Etapas del Proceso

Endpoints relacionados con la configuración y consulta de las etapas elegidas para una requisición, así como las pruebas psicotécnicas, técnicas, entrevistas y assessments asociados.

Lista las etapas elegidas de una requisición

Retorna todas las etapas que fueron seleccionadas (elegidas) para el proceso de selección de una requisición determinada.

Cuando se invoca desde la pantalla de procesos en curso (paraEnProceso=true), se enriquece cada etapa con datos de delegación, conteos de candidatos aprobados/rechazados y el total de candidatos.

El parámetro valOfertaSalarial controla si se aplican reglas de visibilidad sobre la etapa de Oferta Salarial, basándose en la propiedad SLC_OFERTASALARIAL_ETAPA_GESTIONADOR y el permiso PERMISO_SLC_OFERTASALARIAL_MANEJOETAPA.

Authorizations:
bearerAuth
path Parameters
codRequi
required
integer <int64>
Example: 121

Código de la requisición.

query Parameters
paraEnProceso
boolean
Default: false

Indica que la consulta se realiza desde la pantalla de procesos en curso. Si true, se agregan datos de delegación y conteos de candidatos.

valOfertaSalarial
boolean
Default: false

Si true, valida si la etapa de oferta salarial debe ocultarse u ofuscarse según reglas de configuración y permisos del usuario.

Responses

Servidor de producción

https://clientex.talento.cloud/html/etapasElegidas/listar/{codRequi}
